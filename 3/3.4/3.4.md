## 🎯 R014 – Programar inducción

| **Código** | R014 |
|-----------|--------|
| **Objetivo** | Programar una sesión de inducción para el personal nuevo. |
| **Descripción** | El supervisor HSE agenda la inducción considerando fecha, modalidad (virtual/presencial) y personal convocado. |
| **Actor primario** | Supervisor HSE |
| **Actor secundario** | Trabajador |
| **Precondiciones** | Debe existir personal nuevo registrado y una fecha definida para la inducción. |

### 📝 Pasos del caso de uso

| Paso | Acción |
|------|--------|
| 1 | El supervisor accede al sistema. |
| 2 | Selecciona la opción para programar inducción. |
| 3 | Ingresa fecha, hora, modalidad y destinatarios. |
| 4 | Confirma y guarda la programación. |
| 5 | El sistema notifica a los trabajadores asignados. |

---

## 🎯 R015 – Consultar inducciones programadas

| **Código** | R015 |
|-----------|--------|
| **Objetivo** | Visualizar las sesiones de inducción agendadas. |
| **Descripción** | Permite al supervisor HSE consultar las sesiones de inducción filtrando por fecha, modalidad o estado. |
| **Actor primario** | Supervisor HSE |
| **Actor secundario** | - |
| **Precondiciones** | Deben existir sesiones programadas. |

### 📝 Pasos del caso de uso

| Paso | Acción |
|------|--------|
| 1 | El supervisor accede al sistema. |
| 2 | Ingresa al módulo de inducciones. |
| 3 | Visualiza la lista de sesiones programadas. |
| 4 | Filtra por criterios (fecha, modalidad, estado). |
| 5 | Consulta los detalles de cada sesión. |

---

## 🎯 R016 – Editar una inducción programada

| **Código** | R016 |
|-----------|--------|
| **Objetivo** | Modificar los datos de una inducción previamente programada. |
| **Descripción** | Permite cambiar título, modalidad, fecha, contenido o participantes de una inducción. |
| **Actor primario** | Supervisor HSE |
| **Actor secundario** | - |
| **Precondiciones** | Debe existir una inducción registrada para editar. |

### 📝 Pasos del caso de uso

| Paso | Acción |
|------|--------|
| 1 | El supervisor accede al sistema. |
| 2 | Selecciona la sesión a modificar. |
| 3 | Cambia los campos necesarios (contenido, fecha, modalidad, etc.). |
| 4 | Guarda los cambios. |
| 5 | El sistema actualiza la información. |

---

| **Actor primario** | Supervisor HSE |

## 🎯 R018 – Acceder a la inducción asignada

| **Código** | R018 |
|-----------|--------|
| **Objetivo** | Permitir al trabajador visualizar el contenido de inducción. |
| **Descripción** | El trabajador accede al contenido digital (videos, presentaciones, documentos) según la inducción asignada. |
| **Actor primario** | Trabajador |
| **Actor secundario** | Sistema |
| **Precondiciones** | La inducción debe haber sido programada y asignada al trabajador. |

### 📝 Pasos del caso de uso

| Paso | Acción |
|------|--------|
| 1 | El trabajador accede al sistema. |
| 2 | Ingresa a “Mi inducción”. |
| 3 | Visualiza los materiales digitales. |
| 4 | Revisa todo el contenido. |

---

## 🎯 R019 – Rendir evaluación de inducción

| **Código**        | R-I019 |
|-------------------|------------|
| **Objetivo**      | Verificar si el trabajador comprendió correctamente el contenido de la inducción. |
| **Descripción**   | El trabajador, luego de haber visualizado o leído el contenido de la inducción, accede a una evaluación tipo test previamente configurada por el Supervisor HSE. El sistema registra sus respuestas y las corrige automáticamente.|
| **Actor primario**| Trabajador |
| **Actor secundario** | Sistema |
| **Precondiciones**| El trabajador debe haber visualizado el contenido de la inducción. |

---

### 📝 Pasos del caso de uso

| **Paso** | Acción |
|---------|------------|
| 1 | El trabajador accede al sistema e ingresa a su inducción pendiente. |
| 2 | Selecciona la opción para rendir la evaluación. |
| 3 | El sistema carga la evaluación configurada por el Supervisor HSE. |
| 4 | El trabajador responde todas las preguntas del test. |
| 5 | El sistema registra y corrige automáticamente las respuestas. |

---
